**一、项目需求分析**

**1\. 项目背景**

随着移动终端的发展，越来越多的用户希望在繁忙学习与生活之外获得情绪陪伴。  
本项目设计一款以 **可爱的小动物** 为核心的轻互动式情绪陪伴 App，通过 **图像、动画和简单交互** 帮助用户表达每日心情、获得情绪反馈，提升舒适度与趣味性。

**2\. 项目目标**

- 构建一个 **轻量级、小巧精致、面向手机端展示** 的情绪陪伴应用。
- 可以选择自己心仪的小动物
- 通过小动物的动态反馈，让用户在点击、选择心情时获得 **可视化陪伴体验**。
- 提供简单但实用的功能：心情选择、记录历史、收集饰品奖励并兑换得到更多小动物等。
- 最终能以 **Android / iOS 应用形式安装在手机上运行**，方便展示。

**3\. 用户需求分析**

**用户群体**

- 情绪健康关注者
- 喜欢小动物拟人化陪伴的人
- 需要简单、轻松记录心情的学生用户
- 需要可爱的、非严肃型日常记录工具的用户

**用户核心需求**

| **用户需求** | **对应功能** |
| --- | --- |
| 想要快速记录每日心情 | 心情选择器（开心、平静、疲惫等） |
| 想获得轻松的情绪反馈 | 小动物表情与互动动画 |
| 想查看过往心情 | 心情日志（图标 + 日期） |
| 想要激励机制 | 解锁饰品、连续签到奖励 |
| 喜欢可爱、美观、轻交互体验 | 移动端 UI + 动画效果 |

**4\. 功能需求分析（核心）**

**（1）首页——小动物陪伴界面**

- 显示今日小动物形态（根据心情切换图片）
- 点击小动物会出现动态反馈（爱心、小幅跳动）
- 当天心情在顶部显示
- 心情选择区（开心/平静/疲惫…）

**（2）心情记录功能**

- 用户选择心情即自动写入日志
- 数据本地存储（localStorage）
- 可查看过去的心情历史（列表或图标形式）

**（3）饰品奖励系统**

- 用户连续记录心情可解锁饰品
- 用户可给小动物佩戴饰品（鲜花、帽子等）
- 饰品保存在本地

**（4）移动端自适应 UI**

- 采用移动 UI 框架 **Vant**
- 适配不同尺寸手机屏幕
- 使用全屏布局 + 手机触摸交互

**（5）APP 打包能力**

- 使用 **Capacitor** 打包为：
    - Android APK
    - iOS 设备可运行的应用
- 可安装到真机展示
